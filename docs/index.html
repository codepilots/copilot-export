The extension has been vibe coded with claude code, and may still contain bugs. Please use with caution. The extension must be installed with developer mode on in Edge or Chrome as it has not yet been submitted to the store.

Everything you need

Built for full-fidelity export — nothing gets left behind.

📜

Full history

Scrolls through long chats automatically, including lazy-loaded messages, so the entire conversation is captured.

🎨

Rich formatting

Bold, italic, code blocks, headings, tables, and lists are preserved in DOCX and PDF — not stripped to plain text.

🖼️

Inline images

AI-generated images from Microsoft Designer appear exactly where they were in the conversation.

🔗

Citations

Web references are exported as clickable hyperlinks in every format that supports them.

🔒

Fully local

No data ever leaves your machine. Export happens entirely in your browser — no server, no account.

⚡

No build step

Pure vanilla JavaScript. The DOCX builder, ZIP assembler, and Open XML generator are all in-browser.

Install the extension

Choose the method that works best for you.

Recommended

⚡ One-command install

Paste this into PowerShell. Edge downloads and installs the extension automatically — no developer mode needed.

irm 'https://codepilots.github.io/copilot-export/install.ps1' | iex
  1. Press Win + X → Windows PowerShell
  2. Paste the command above and press Enter
  3. If prompted, allow Edge to restart
  4. Visit copilot.microsoft.com and click the extension icon

🔧 Manual install

Download the source and load it yourself — ideal if you want to inspect or modify the code first.

⬇ Download ZIP
  1. Unzip the downloaded file
  2. Open edge://extensions in Edge
  3. Enable Developer mode (top-right toggle)
  4. Click Load unpacked and select the unzipped folder
🔒
Privacy: The extension only activates on Microsoft Copilot pages. It reads your chat DOM to build the export file — everything stays on your device. No analytics, no telemetry, no remote calls except fetching your own generated images (to embed them in exports). Review the source code any time.
